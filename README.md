# BtnCpp
⚡ C++ IDE Gadget: Generate code with just a few button clicks ⚡
⚡ Very rudimentary code ⚡ A practice project when bored ⚡

## Compilation Notes
If using g++, create a "mingw64" folder in the same directory as the compiled executable,
and place the g++ suite directly inside (cppbuild.exe)

## Project Background
One day, I got tired of typing code and thought: could I generate templates with just a few button clicks?
So I invented BtnCpp. Although I used AI assistance (DeepSeek) to generate some parts,
such as code highlighting and other features were AI-assisted.

## Usage Instructions

For example:

1. Find the button labeled "main" (roughly in the center)
2. Click it
3. A pop-up asks if you want parameters
4. Bam! The following appears on the right:

```cpp
int main(int argc, char* argv[]) { // Main function body
    return 0;
}
```
# BtnCpp
⚡ 小工具 C++ IDE：点几下按钮就能生成代码 ⚡ 
⚡ 代码非常简陋 ⚡ 闲着没事练手项目 ⚡ 

## 编译说明
如果使用 g++，在编译后的可执行文件同级目录下新建 mingw64 文件夹，
里面直接放 g++ 套件即可（cppbuild.exe）

## 项目背景
一天，我敲代码敲烦了，想着：能否点几下按钮自动生成模板？
于是，我就发明了 BtnCpp。虽然我用了 AI 辅助生成（DeepSeek），
比如代码高亮等功能都是 AI 辅助生成的。

## 使用方法

例如：

1. 找到写着 "main" 的按钮（大概在中间位置）
2. 点一下
3. 弹出个框问你要不要参数
4. 啪！右边就出现了：

```cpp
int main(int argc, char* argv[]) { // 主函数体
    return 0;
}
```
